


Notice: User may request task/create for multiple times.

If posts with the same platform and external_id is requested multiple times within  1 day (or a week, can be configured), the post will not be updated. If requested over 1 day, you should update that post. Here are some possible changes:
1. Post title/description change
2. Author name/avatar change.
3. Media order change (media in the first is moved to the last, etc.)
4. Some of the media is deleted.
5. Media url/type update, ...

(post can have no external id, post with no external id should be considered as different posts)
